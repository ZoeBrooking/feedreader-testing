# feedreader-testing

## Feedreader Testing for Udacity Front-End Developer Nanodegree

### How to run:

1. Clone the github directory.
2. Open index.html.
3. The tests will run automatically upon opening.
4. The results of the tests will be at the bottom of the webpage.

### Dependencies:

* jQuery v 3.3.1
* Jasmine v 3.3.0
* Handlebars v 4.0.12
* Google JSAPI
